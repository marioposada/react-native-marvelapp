# Description
Appmarvel is an App coded in React Native using the Marvel API.

## Build Setup

``` bash
# clone project
$ git clone git@github.com:marioposada/react-native-marvelapp.git

# install dependencies
$ npm install

# serve with host at localhost:19000
$ npm start
$ yarn start
```
